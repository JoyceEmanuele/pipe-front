# ToggleSwitchMini

Componente que permite alterar entre dois valores de forma exclusíva, simulando um funcionamente de um interruptor (toggle switch), mas
tendo tamanho menor que o ToggleSwitch.

A propriedade `checked` é obrigatória.

A propriedade `onOff` é opcional.

A propriedade `disabled` é opcional.

A propriedade `onClick` é opcional.

## Propriedades:

`checked`: False -> Switch desmarcado; True -> Switch marcado

`onOff`: False -> Comportamento normal do componente; True -> Switch passa a considerar que quando checked for true, significa que o valor está desligado,
                  portanto a cor da bola presente no componente será preta.

`disabled`: Desabilita o componente

`onClick`: Recebe o evento ao clicar no Switch
