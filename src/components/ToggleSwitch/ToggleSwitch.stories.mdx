# ToggleSwitch

Componente que permite alterar entre dois valores de forma exclusíva, simulando um funcionamente de um interruptor (toggle switch).

A propriedade `checked` é obrigatória.
As propriedades `label`, `variant` e `sLabel` são opcionais. A propriedade sLabel tem efeito apenas quando o variant for 'bloq'

Propriedades:
`checked`: False -> Switch desmarcado; True -> Switch marcado
`label`: Label à direita de component
`variant`: auto -> Label dentro de componente com valores AUTOMÁTICO e MANUAL dependendo do checked;
           bloq -> Label dentro de componente com valores BLOQUEADO e LIBERADO dependendo do checked;
`sLabel`: Label dentro de componente para quando o `variant` for 'bloq'
